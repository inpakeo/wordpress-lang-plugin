# SVG Flags Directory

This directory is for SVG flag images.

## Naming Convention

Name files using ISO 2-letter country codes in lowercase:
- `en.svg` - English/United Kingdom
- `us.svg` - United States
- `fr.svg` - France
- `de.svg` - Germany
- `es.svg` - Spain
- `ru.svg` - Russia
- `cn.svg` - China
- etc.

## Usage

The plugin will automatically use SVG flags if they exist, otherwise it will fall back to emoji flags.

SVG flags are preferred for:
- Better scaling
- Consistent appearance across browsers/OS
- Professional look

## Recommended SVG Flag Sets

You can download free SVG flags from:
- https://github.com/lipis/flag-icons (Free MIT license)
- https://flagicons.lipis.dev/
- Country Flags: https://github.com/hjnilsson/country-flags
